#  Ecommerce Sales Analysis (Power BI)

##  Overview

This project presents an **interactive Power BI dashboard** to analyze ecommerce sales data and extract meaningful business insights.

---

##  Features

* Visualizes sales trends and revenue
* Analyzes profit and category performance
* Tracks customer and regional insights
* Interactive dashboard with filters

---

##  Tools Used

* Power BI
* Data Visualization
* Data Cleaning & Analysis

---


##  Insights

* Top-performing products and categories
* Revenue and profit trends over time
* Regional sales distribution
